# Employee Spring

A simple Spring Boot project for Employee Management.

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## Technologies Used
- Java
- Spring Boot
- Maven
- MySQL
- Spring Data JPA

## Prerequisites
- Java 17+
- Maven
- MySQL

## Run the Project

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Open the project in your IDE.

3. Configure the database in `application.properties`.

4. Run the project:
   ```bash
   mvn spring-boot:run
   ```

5. Open in browser:
   ```
   http://localhost:8080
   ```
